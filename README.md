# 🚀 LeetCode Solutions

A personal collection of my solutions to various [LeetCode](https://leetcode.com/) problems. This repository serves as a log of my algorithmic problem-solving journey. 

## 🛠️ Languages & Tools

The primary programming language used in this repository is **Kotlin**, alongside some Shell scripts for utility tasks.

## 📂 Repository Structure

All problem solutions and related source code are organized within the `src/` directory.

```text
LeetCode/
├── src/
├── .gitignore
└── README.md
```

## 🏃‍♂️ How to Run

You can open the project in any standard Kotlin IDE (like IntelliJ IDEA) or run the files via the command line.
